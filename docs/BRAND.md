# Bridge - Brand Identity

## Logo Concept: The Card Bridge

The Bridge logo is a visual play on two meanings:
1. **A physical bridge** - connecting networks securely, spanning gaps
2. **The card game Bridge** - strategy, partnerships, trust, and calculated play

### Logo Description

The primary logo depicts a **bridge structure built from playing cards** - cards leaning against each other to form the classic card house bridge/arch shape. This creates a visual that is:

- Immediately recognizable as both a bridge AND cards
- Suggests both **strength** (engineering, structure) and **strategy** (the card game)
- Implies **precision** - a card bridge requires careful placement, just like zero-trust security
- The arch shape naturally evokes **connection** between two sides

### Visual Concept

```
        ┌───┐
       /│ ♠ │\
      / └───┘ \
   ┌───┐     ┌───┐
   │ ♦ │     │ ♣ │
   └───┘     └───┘
  /     \   /     \
━━━━━━━━━━━━━━━━━━━━━
```

**Primary mark:** Two playing cards leaning together to form an arch/bridge span. The cards show suit symbols (spades, diamonds, clubs, hearts) that double as visual metaphors:

| Suit | Security Metaphor |
|---|---|
| ♠ Spades | Shield / protection (the spade shape resembles a shield) |
| ♦ Diamonds | Encryption / value (protecting what matters) |
| ♣ Clubs | Network / connection (the three lobes suggest nodes) |
| ♥ Hearts | Trust / integrity (the core of zero-trust) |

### Logo Variations

1. **Full logo:** Card bridge arch + "BRIDGE" wordmark
2. **Icon only:** Card bridge arch (for app icons, favicons)
3. **Simplified:** Two overlapping cards forming a "B" shape (for very small sizes)
4. **Monochrome:** Single color version for dark/light backgrounds

### Color Palette

| Name | Hex | Usage |
|---|---|---|
| Bridge Navy | `#1B2A4A` | Primary brand color, dark backgrounds, wordmark |
| Card Red | `#C41E3A` | Accent, hearts/diamonds on cards, alerts/warnings |
| Trust Green | `#2ECC71` | Connected/healthy status, posture passing |
| Signal Blue | `#3498DB` | Interactive elements, links, tunnel indicators |
| Paper White | `#F5F5F0` | Card faces, light backgrounds, clean space |
| Felt Green | `#1A7A3A` | Optional: nod to card table felt, used sparingly |

### Typography

- **Wordmark:** A clean, geometric sans-serif that conveys precision and modernity
  - Candidates: Inter, Satoshi, Plus Jakarta Sans, or custom
- **Body text:** Inter or system font stack for UI
- **Code/technical:** JetBrains Mono or similar monospace

### Tagline Options

- "Every connection, secured." - straightforward, confident
- "Trust nothing. Bridge everything." - zero-trust philosophy
- "Your network. Your rules." - empowerment angle
- "Play your cards right." - playful nod to the card game theme
- "Bridging trust in a zero-trust world." - explains the paradox of the product

### App Icon Concept

The app icon (macOS menu bar, iOS home screen, Android, Windows taskbar) should be:
- The card bridge arch in simplified form
- Recognizable at 16x16 pixels (menu bar) through 1024x1024 (App Store)
- Two cards leaning together, with a subtle suit symbol visible
- Bridge Navy background with Paper White cards (or inverted for light themes)

### Brand Voice

| Attribute | What It Means |
|---|---|
| **Confident** | We know security. No hedging, no "may help protect." |
| **Clear** | Technical without being jargon-heavy. Explain, don't obfuscate. |
| **Direct** | Short sentences. Active voice. Say what we mean. |
| **Trustworthy** | Transparent about what we inspect and what we don't. |
| **Strategic** | Like the card game - every move is calculated and intentional. |

### Card Game Terminology in the Product

The card game metaphor can extend into the product itself:

| Card Game Term | Product Usage |
|---|---|
| **Hand** | A device's current posture state ("your hand is strong" = good posture) |
| **Trick** | A completed security check cycle |
| **Trump** | Override policy (admin can "trump" a default rule) |
| **Bid** | Access request ("the device bid for Salesforce access") |
| **Partner** | The relay (your trusted partner across the bridge) |
| **Dummy hand** | Transparent mode (you can see exactly what Bridge is doing) |
| **Grand slam** | Perfect posture score (100/100) |
| **Vulnerable** | Device with failing posture checks |

Use sparingly - the metaphor should enhance, not confuse. Keep it subtle in the UI, more prominent in marketing.
